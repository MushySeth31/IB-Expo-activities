# Order System

## Process
1. User pushes any of the item buttons
2. User pushes the confirm button
3. Wait 10 seconds, red LED turns on
4. After 10 seconds, buzzer alerts, red LED turns on, green LED turns on
5. User pushes the confirm button to claim (process repeats)
